# 🏛️ NMC Complaint Box

A Complaint Management System developed using **Java, Spring Boot, Thymeleaf, MySQL, Bootstrap, HTML, CSS, and JavaScript**. This application allows citizens to submit complaints online while administrators can efficiently manage, review, and resolve complaints.

---

# 📖 About the Project

The **NMC Complaint Box** is a web-based application designed to simplify the complaint registration and management process. Users can submit complaints with supporting documents, while administrators can track, update, and resolve complaints through an organized dashboard.

# 🚀 Installation

### 1. Clone Repository

```bash
git clone https://github.com/your-username/NMCComplaintBox.git
```

### 2. Open Project

Import the project into Eclipse or IntelliJ IDEA.

### 3. Configure Database

Create a MySQL database and update the following properties in:

```
application.properties
```

```
spring.datasource.url=jdbc:mysql://localhost:3306/database_name

spring.datasource.username=root

spring.datasource.password=your_password
```

### 4. Run Project

```
mvn spring-boot:run
```

or

Run the project from your IDE.
